Place your profile image file here so Vite will serve it at the root URL path.

Steps:
1. Copy the image file from:
   c:\Users\aryan\Downloads\compressed_image.jpg

2. Paste it into this folder:
   <project-root>/sportsy/public/

3. Rename the file to:
   compressed_image.jpg (if it's not already named that)

4. Start the dev server (if not running):
   npm run dev

The image will then be available at:
   http://localhost:5173/compressed_image.jpg

And the app will reference it as `/compressed_image.jpg` for the profile picture.
